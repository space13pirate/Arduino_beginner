Eng: Description of the smart home project "Doorbell".
When the button is pressed, the doorbell melody is played. The melody can be changed.
A buzzer and a standard button are used.

Рус: Описание проекта умного дома "Дверной звонок".
При нажатии кнопки воспроизводится мелодия дверного звонка. Мелодию можно менять.
Используется пьезоизлучатель и стандартная кнопка.
